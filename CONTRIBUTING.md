# Contributing to NEXUM Protocol

We welcome contributions from the SEA Web3 community! 🌏

## Ways to Contribute

- 🐛 Report bugs
- 💡 Suggest features
- 🔧 Submit pull requests
- 📖 Improve documentation
- 🧪 Write tests

## Getting Started

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/nexum-protocol-mvp`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make changes and test
5. Push and open a Pull Request

## Code Style

- Rust: follow `rustfmt` standards
- TypeScript: ESLint + Prettier
- Commit messages: `feat:`, `fix:`, `docs:`, `test:`
